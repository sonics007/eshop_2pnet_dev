# 2Pnet E-shop

Moderný e-shop vytvorený v Next.js 14 + Tailwind CSS pre 2Pnet s.r.o. Obsahuje landing stránku, dynamické produktové stránky, košík aj kontaktný formulár.

## Požiadavky
- Node.js 18+
- npm alebo yarn/pnpm

## Inštalácia
```bash
npm install
npm run dev
```

Potom otvorte [http://localhost:3000](http://localhost:3000).

Projekt je pripravený na rozšírenie o CMS, reálne API a platobnú bránu.
